# 股票策略回测系统 - 前端

基于Vue 3开发的股票K线图展示系统，支持分钟级、日级、周级、月级数据展示。

## 功能特性

- 📈 实时K线图展示（类似同花顺界面）
- 🔍 股票代码查询（支持格式：000001.SZ 或 000001）
- ⏱️ 多周期切换（分钟、日线、周线、月线）
- 📊 技术指标：MA5、MA20移动平均线
- 📉 成交量柱状图
- 🎨 深色主题界面

## 技术栈

- Vue 3
- Vite
- ECharts 5
- Axios

## 安装和运行

### 1. 安装依赖

```bash
cd vue
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

前端将在 `http://localhost:3000` 启动。

### 3. 确保后端服务运行

确保Flask后端服务在 `http://localhost:8080` 运行：

```bash
cd ..
python app.py
```

## 使用说明

1. **输入股票代码**：在顶部输入框输入股票代码（如：000001.SZ），按回车或点击查询按钮
2. **切换周期**：点击顶部的"分钟"、"日线"、"周线"、"月线"按钮切换不同的时间周期
3. **查看图表**：
   - K线图：红色表示上涨，绿色表示下跌
   - MA5/MA20：移动平均线
   - 成交量：底部柱状图
4. **缩放和拖拽**：使用底部滑块或鼠标滚轮缩放和拖拽查看历史数据

## API接口

前端通过以下API与后端通信：

- `GET /api/stock/<stock_code>?period=<period>` - 获取股票数据
  - `stock_code`: 股票代码（如：000001.SZ）
  - `period`: 时间周期（minute/day/week/month）

## 构建生产版本

```bash
npm run build
```

构建后的文件将输出到 `dist` 目录。

## 注意事项

- 确保后端Flask服务已启动并运行在8080端口
- 数据文件应存放在上层目录的 `data` 文件夹中
- 股票代码格式：`000001.SZ`（深市）或 `000001.SH`（沪市）
